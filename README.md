# TechFlowSolutions
Sistema de login (Trabalho acadêmico)

Este projeto é sobre a criação de um sistema de gerenciamento de tarefas baseado em metodologias ágeis para a empresa TechFlow Solutions. Para seu desenvolvimento serão utilizadas ferramentas como o GitHub, um quadro Kanban dentro do GitHub e o VS Code. O GitHub será utilizado como repositório do projeto, o quadro Kanban será utilizado para organizar as tarefas e monitorar seu progresso, enquanto o VS Code será utilizado para escrever o código e enviar para o repositório remoto.
